## Tools Used
- Python
- Pandas
- Matplotlib
- SQL

## Analysis Performed
- Total sales & profit by region
- Category-level and regional profitability
- Data visualizations (7 plots)

## Outcome
- Extracted business insights to identify high-performing regions and categories
